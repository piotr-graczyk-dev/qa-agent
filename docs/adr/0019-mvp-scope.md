# MVP scope

The MVP for the v1 QA Agent is an Expo/EAS CI Runner QA Agent that can initialize an Android-first workflow, validate configuration, run through Eve with `agent-device`, perform secret-isolated email/password Agent Login, capture artifact screenshots, write a validated QA Report, and upsert one GitHub pull request comment. Hosted mode, additional login providers, inbox adapters, extra storage providers, non-EAS CI, deterministic test DSLs, and additional mobile automation backends are outside the MVP.
